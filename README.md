# Setupping project

### Install Dependencies

```bash
npm install
```

### Create a .env file by copying the existing one:

```bash
cp .env.sample .env
```

### Set the value of REACT_APP_BASE_URL pointing to back-end API mock

http://www.mocky.io/v2/5eb553df31000060006994a8

### Run the project

```bash
npm run start
```
